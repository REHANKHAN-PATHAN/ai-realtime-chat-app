# AI Real-Time Chat App

A modern full-stack real-time AI-powered chat application built using the MERN stack and Socket.io.

## Features
- Real-time messaging
- Group chat
- Voice messaging
- File sharing
- AI integration
- Responsive UI
- Secure authentication

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Socket.io
